# QuestGen AI - Frontend

AI-based automated exam question paper generation and CO-PO mapping system frontend application.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

## Technologies Used

- React 18.2.0
- Bootstrap 5.3.2
- React Router DOM 6.20.0
